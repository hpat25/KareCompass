KareCompass (HackKU 2026)

KareCompass is an interactive web application built during HackKU 2026 that helps users find and compare healthcare providers based on cost, accessibility, and insurance compatibility. By combining real-time data, scoring algorithms, and AI assistance, KareCompass turns complex healthcare decisions into simple, informed choices.

Features: 
Interactive Map: Select a county to view available healthcare providers
Smart Pins: Instantly compare providers by cost directly on the map
Side Panel Insights: View affordability, accessibility, and overall scores
Personalized Results: Input insurance to optimize provider rankings
AI Assistant: Get simple explanations of provider data and insurance concepts

Tech Stack:
Frontend: React, JavaScript, HTML, CSS
Backend/Environment: Node.js
Design: Figma → SVG integration
AI: Google Gemini API
Version Control: Git & GitHub

Running the Project:
git clone https://github.com/hpat25/KareCompass.git
cd KareCompass
npm install
npm start
